# Animate UI Animated Icons (beta)

260 animated Lucide icons. Install the wrapper first, then any icon:

```bash
npx shadcn@latest add @animate-ui/icons-icon        # required wrapper
npx shadcn@latest add @animate-ui/icons-<lucide-name>
```

Usage: `<ArrowRight animateOnHover />` or wrap several icons with `<AnimateIcon animateOnHover>...</AnimateIcon>`.
